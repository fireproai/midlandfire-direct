Morley / Addressable Interface modules / All

Trade-only SKU list for this range, ordered A-Z to support specification and replacement sourcing.

Alphabetical SKU list

  • 797-065 - Multi-Mount Enclosure. Metal, Morley light grey
  • CMX-10RME - 10-way relay control (output) card (unboxed PCB.) contact rating 30 Vdc, 1 A. Complete with individual input disable switches, plug in screw connections and on board LED's with the facility for external LED's
  • CR-6EA - six way changeover relay output with built-in loop isolation. Complete with rotary address switch and plug-in terminal blocks. Requires M200-SMB-MM for mounting
  • DRM-1 - BS7273 Door Release Module, additional failsafe to be used in conjunction with Morley-IAS output modules to provide a compliant solution to BS7273-4 Category A requirements. Powered from the 12-24v door magnet supply
  • IM-10EA - Ten input monitor module to provide an interface to a normally open contact devices such as plant outputs or flow switches. Requires back box 797-065
  • M200E-DIN - DIN rail mounting clip for M2xx series modules
  • M200E-SMB - Surface mounting box for M2xx series modules
  • M200SMB - Deep Module enclosure
  • M200XE - Loop isolation module. Requires M200E-SMB for surface mounting or M200E-DIN for mounting to DIN rails
  • M201EA - SINGLE OUTPUT MODULE - SYSTEM SENSOR
  • M201EA-240 - Output control module System Sensor Protocol. 240V ac relay contact rating, no longer comes with an enclosure e.g. M200E-SMB
  • M210EA - SINGLE INPUT MODULE - SYSTEM SENSOR
  • M210EA-CZ - Analogue addressable monitor module for supervision of a conventional zone equipped with capacitive EOL element
  • M210EA-CZR - Analogue addressable monitor module for supervision of a conventional zone equipped with EOL resistor
  • M220EA - DUAL INPUT MODULE - SYSTEM SENSOR
  • M221EA - 2 INPUT, 1 OUT MODULE - SYSTEM SENSOR
  • M501MEA - Addressable "mini" input module. It will interface with contact devices such as security contacts and waterflow switches, suitable for mounting inside monitored equipment
  • MCX-55ME - 5-way Addressable monitor (input) & 5 way relay control (output) card (unboxed) complete with end of line resistors, individual input disable switches, plug in screw connections and on board LED's with the facility for external LED's. Requires back box 797-065
  • MMX-10ME - 10 way Addressable monitor (input) card (unboxed) complete with end of line resistors, individual input disable switches, plug in screw connections and on board LED's with the facility for external LED's. (Use only as spares/replacements and for extensions of existing sites)
  • SMB500 - Surface Mount Module Box. Plastic, not suitable for M7xx modules
  • SMB6-V0 - Surface mount box for up to 6 M2xx modules
  • SSE-DRS-1 - BS7273 Door Release system. Comprising DRM-1 door release module and M201 single output model prewired and mounted in SMB6-V0 surface mount box. Used provide a compliant solution to BS7273-4 Category A requirements. Powered from the 12-24v door magnet supply. System Sensor protocol
